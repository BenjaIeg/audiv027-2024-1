## crear carpeta para ti en tu fork

- ir a tu fork
- comprobar que estás en tu fork
- hacer click en botón "create new file"
- escribir tu nombre de usuario de github seguido de un slash
- escribir "README.md"
- escribir en la línea 1 "hola mundo"
- hacer commit
